# MindHub

Website personal all in one.

## Features
- To do list
- Document Management
- Study Tracker
- News 
- Stock News
- Gallery
- Notes

## Installation
1. Clone repo
2. Install dependencies
3. Run the app

## Usage
Cara make project-nya:
```bash
npm start
```
